# 🚗 Driver Safety Monitoring System

## 🔍 Overview
An AI-based system that detects driver drowsiness, chest pain, and unsafe behavior using computer vision.

## ⚙️ Tech Used
- Python
- OpenCV
- MediaPipe
- Flask

## 🚀 Features
- Eye closure detection
- Head drop detection
- Chest pain detection
- Alarm system

## 📌 Future Improvements
- Mobile app integration
- Emergency alert system
